Ketentuan Layanan (Terms of Service) - rsynzr Discord Bot

1. Penerimaan Ketentuan

Dengan mengundang atau menggunakan rsynzr Discord Bot di server Discord Anda, Anda menyetujui Ketentuan Layanan ini. Jika Anda tidak setuju, jangan tambahkan atau gunakan bot ini.

2. Deskripsi Layanan

Bot ini menyediakan fitur cuaca, notifikasi, dan utilitas lain yang berinteraksi dengan API cuaca pihak ketiga. Bot berjalan di server kami dan berkomunikasi dengan API serta Discord untuk memenuhi permintaan pengguna.

3. Penggunaan API Cuaca

Kami menggunakan layanan API cuaca pihak ketiga untuk mengambil informasi cuaca. Penggunaan API tersebut tunduk pada kebijakan dan persyaratan penyedia API. Dengan menggunakan bot ini, Anda setuju bahwa data cuaca disediakan "apa adanya" dan kami tidak bertanggung jawab atas ketidakakuratan atau keterlambatan.

4. Izin dan Data

Bot mungkin memerlukan izin tertentu (misalnya membaca saluran, mengirim pesan) dan menyimpan beberapa informasi sederhana seperti konfigurasi server. Kami tidak mengumpulkan atau menyimpan data pribadi sensitif.

5. Batasan Penggunaan

  • Jangan menyalahgunakan bot untuk spam atau tindakan yang melanggar Ketentuan Discord.
  • Tidak diizinkan menggunakan bot untuk tujuan ilegal, termasuk namun tidak terbatas pada penyebaran malware, penipuan, atau pelanggaran hak cipta.
  • Jangan memanipulasi atau melebihi batasan API cuaca atau Discord.

6. Iklan dan Monetisasi

Bot ini tidak menampilkan iklan secara langsung. Namun kami dapat memonetisasi proyek melalui situs web atau layanan lain yang dapat mencakup iklan AdMob, AdSense, atau afiliasi.

7. Hak Milik Intelektual

Kode sumber, merek dagang, dan konten bot adalah milik rsynzr atau lisensor kami. Anda diberi lisensi tidak eksklusif untuk menggunakan bot sesuai Ketentuan ini.

8. Penyangkalan Jaminan

Bot disediakan "sebagaimana adanya" tanpa jaminan apa pun. Kami tidak menjamin bot akan selalu berfungsi atau bahwa data cuaca akan akurat.

9. Batasan Tanggung Jawab

Kami tidak bertanggung jawab atas kerugian langsung, tidak langsung, khusus, atau konsekuensial yang timbul dari penggunaan bot atau dependensinya pada API cuaca.

10. Perubahan Ketentuan

Kami dapat memperbarui Ketentuan ini kapan saja. Perubahan akan dipublikasikan di halaman ini dan penggunaan bot setelah perubahan dianggap sebagai penerimaan terhadap Ketentuan terbaru.
